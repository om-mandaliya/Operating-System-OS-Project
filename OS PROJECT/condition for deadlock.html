CONDITION FOR DEADLOCK


  • There for condition that must hold for a deadlock to occur. these conditions are as follows:-

    1. MUTUAL EXCLUSION CONDITION:

      2. Only one process can use a remove at any given time.

    2. HOLD AND WAIT:

      3. A process is holding at least one resorce at a time and is waiting to acquire other resources held by some other process.

    3. NO PREEMPTION:

      4. The resource can be released by a process voluntarily.

    4. CIRCULAR WAIT

      5. A set of process are waiting for each other in a circular fashion
